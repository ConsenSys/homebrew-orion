# homebrew-orion
Homebrew Tap for Orion.

## Getting started
Add the tap, and install:

```
brew tap pegasyseng/orion
brew install orion
```
Run the orion executable:

```
orion --help
```

Refer to the [orion docs](https://docs.orion.pegasys.tech/en/stable/) for further information.

To upgrade orion:
```
brew tap pegasyseng/orion
brew upgrade orion
```
