# homebrew-orion
Homebrew Tap for Orion.

## Getting started
Add the tap, and install:

```
brew tap consensys/orion
brew install orion
```
Run the orion executable:

```
orion --help
```

Refer to the [orion docs](https://docs.orion.consensys.net/) for further information.

To upgrade orion:
```
brew tap consensys/orion
brew upgrade orion
```
